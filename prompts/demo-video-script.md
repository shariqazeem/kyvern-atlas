# Demo Video Script — Stellar Hacks: Agents
## "Pulse — Revenue Intelligence for x402 on Stellar"

**Length:** 2:30
**Style:** Screen recording + voiceover. No face cam. Calm, confident, conversational.
**Tool:** QuickTime Player → File → New Screen Recording (or Loom)

---

## Setup Before Recording

Open these tabs in order:
1. `kyvernlabs.com/pulse/dashboard` (logged in, Overview page)
2. `kyvernlabs.com/pulse/dashboard/transactions` (Network filter set to "All networks")
3. `kyvernlabs.com/pulse/dashboard/copilot`
4. `kyvernlabs.com/pulse/dashboard/setup` (Stellar tab selected)
5. Keep one empty tab ready for StellarChain.io

Pre-check:
- Dark mode OFF (white theme)
- Dashboard has fresh data (both Base and Stellar transactions visible)
- Browser bookmarks bar hidden (Cmd+Shift+B in Chrome)
- Font is readable at 1080p
- Test mic — speak and listen back

2. Trim dead air at the start/end.
3. Export at 1080p minimum.
4. Upload to YouTube (unlisted) or Loom.
5. Add the link to HACKATHON.md and the DoraHacks submission.
6. Submit at least 4 hours before deadline.

---

## Key Tips

- **Slow mouse movements.** Don't zip around the screen.
- **Pause on the StellarChain.io page.** That's the "whoa moment" — let judges read it.
- **Don't apologize or say "um."** If you stumble, just pause and continue.
- **Energy matters.** Sound confident, not bored. You built something real.
- **Under 2:30.** Judges skip anything over 3 minutes.
